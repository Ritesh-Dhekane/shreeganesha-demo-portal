# Client Approvals Archive

This directory stores formal client approvals, sign-offs, and feedback summaries for Shree Ganesha demos.

## Purpose
* Maintain a historical record of what was approved and when.
* Store email confirmations, chat screenshots, or PDF sign-off documents.
* Reference approval milestones before moving features to production.

## File Naming Convention
* Use format: `YYYY-MM-DD-demo-XXX-approval.md` (e.g. `2026-06-15-demo-001-approval.md`).

## Workflow
1. Share the demo portal URL with the client.
2. Receive confirmation of approval on the design/concept.
3. Record the approval details in this directory.
